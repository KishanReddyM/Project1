3. Anchor Tags with 'rel' Attributes

These links have different 'rel' attributes that tell search engines and browsers how to handle them.

  • nofollow: Tells search engines not to follow this link.
  • noopener: Prevents the new page from accessing window.opener.
  • noreferrer: Prevents sending the referrer header to the new page.
